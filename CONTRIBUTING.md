# Contributing to Awesome Radio

Please note that this project is released with a [Contributor Code of Conduct](https://www.contributor-covenant.org/version/2/1/code_of_conduct/). By participating in this project you agree to abide by its terms.

---

## Guidelines

- **One item per pull request.** This makes it easier to review and merge.
- **Check that your suggestion isn't already on the list.**
- **Make an individual pull request for each suggestion.**
- **Use the following format:** `- [Name](link) - Description starting with capital letter, ending with period.`
- **Descriptions must not be promotional.** They should describe what the resource is, not how great it is.
- **New categories** are welcome but only if you have at least 3 items to populate them.
- **Check your spelling and grammar.**
- **The pull request and commit must have a useful title.**
- **Links must resolve.** The CI will check all links via awesome_bot.

## Entry Format

```
- [Resource Name](https://example.com) - What it does, in one sentence.
```

- Starts with capital letter
- Ends with a period
- No trailing whitespace
- No duplicate links
